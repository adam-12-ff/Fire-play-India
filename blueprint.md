# **App Name**: FirePlay Arena

## Core Features:

- Auth (Authentication System): User Login/Register via phone, name, password. 2-step signup with gaming info (UID, Level, Username). Firebase-based secure auth.
- User Profile: Show profile pic (optional), UID, rank, level. Wallet balance visible. Profile edit feature.
- Home Page (Dashboard): Modes: BR & CS. Room filters: 1v1 to 4v4. Shows joinable rooms with status (full/open). Join/Create Room options.
- Create Tournament (Host Panel): Build a 5-step responsive Tournament Creator for “FirePlay”. Access only for users with Verified FirePlay Card & no active tournaments.
- Join Tournament (Player Panel): Enter Room ID & Password. Verify available slot and deduct entry fee. Auto add to Room Participant List. Show success/failure message.
- Match Status & History: Live Status: Starting, Ongoing, Completed. Result Upload by Host. Result Approval by Admin. Win/Loss history in profile. Points & Stats Update.
- Wallet System: Entry Fee Deduction. Winnings Credit. Withdraw Request option (UPI-based). Admin Approval for Withdrawals.
- Notifications: Important alerts: Match start time, status updates. Custom admin broadcast to all users.
- Admin Panel (Web Only): Room create/delete/edit. User data monitor. Match status change. Withdraw approval. Ban/kick users.
- Support System: Chat/Message section. WhatsApp click-to-contact. FAQ page for help.
- Smart Features (Optional): AI-based auto room naming. Predictive tournament suggestions. OCR-based UID verification (from screenshot).

## Style Guidelines:

- Primary color: Vibrant Purple (#A020F0) to evoke excitement and a sense of digital competition.
- Background color: Dark gray (#222222) to enhance readability and reduce eye strain during long gaming sessions.
- Accent color: Electric Green (#7CFC00) for highlights, calls to action, and to reflect the energy of esports.
- Font: 'Space Grotesk' (sans-serif) for both headings and body text, giving a computerized and techy look suitable for an e-sports arena.
- Use of neon-style icons that glow against the dark background, enhancing the modern gaming look. The icons will represent various game modes, user actions, and in-game items.
- Mobile-friendly layout with a focus on quick access to tournaments and wallet information. Utilize card-based UI for tournament listings and a clear, easy-to-navigate menu.
- Subtle animations on button presses and data loading to enhance user engagement. A slight glow or pulse effect on interactive elements will indicate their active state.
